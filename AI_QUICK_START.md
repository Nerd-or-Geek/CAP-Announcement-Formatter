## Troubleshooting

### Ollama Won't Install
- Download manually: https://ollama.com/download
- Run installer
- Restart computer
- Try setup again

### Setup Stuck at "Downloading"
- Check internet connection
- Make sure you have 5GB+ free disk space
- Try again or choose smaller model

### Button Greyed Out
- Make sure you selected a widget (click in canvas)
- Widget must have content to improve
- Try refreshing by selecting another widget

### AI Takes Too Long
- First request loads model (slower)
- Subsequent requests are faster
- Close other applications
- Check your system meets requirements
